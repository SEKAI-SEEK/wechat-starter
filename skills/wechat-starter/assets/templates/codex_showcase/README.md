# Project Progress

This folder contains only compact, reusable project decisions and preview evidence created during the WeChat Starter workflow.

## Files

- `project-brief.md`: approved first-preview scope.
- `stage-records.md`: current stage and passed gates.
- `evidence-ledger.md`: observable proof and unresolved evidence.
- `preview-checklist.md`: final WeChat DevTools acceptance.

Keep long reasoning in the conversation. Never store credentials, private configuration, personal data, login QR codes, or unredacted account screenshots here.
